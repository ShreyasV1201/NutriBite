# NutriBite

NutriBite is a personalized food recommendation platform that suggests suitable meals based on user health conditions and dietary preferences using machine learning.

The project aims to help users make healthier nutritional choices through intelligent meal recommendations powered by data analysis and predictive modeling.

---

## Features

* Personalized food recommendations
* Health condition–based meal suggestions
* Machine learning powered prediction system
* User-friendly web interface
* Nutritional data analysis
* Cross-platform accessibility

---

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Flask (Python)

### Machine Learning & Data Processing

* Python
* Pandas
* Scikit-learn
* Decision Tree Classifier

---

## Machine Learning Workflow

1. User enters health-related information and dietary preferences
2. Dataset is processed and analyzed
3. Machine learning model predicts suitable food options
4. Personalized meal recommendations are displayed to the user

## Installation

Clone the repository:

```bash
git clone https://github.com/ShreyasV1201/NutriBite.git
```

Move into the project directory:

```bash
cd NutriBite
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Flask server:

```bash
python app.py
```
